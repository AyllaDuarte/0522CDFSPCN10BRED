CSS: Seletores

Id: #nome-id {}

Seleção por id deve ser única, ou seja, utilize o nome da id em apenas um elemento.

#aplicativo {}

<div id="aplicativo">...</div>

Classe: .nome-class {}

Seleção por class pode ser utilizada sem restrições, ou seja, é perfeito para reaproveitamento de código.

.componente {}

<article class"componente">...</article>

Elemento: a {}

Seleção por elemento deve ser utilizado com cautê-la, pois personaliza todos os elementos de sua seleção.

a { text-decoration: none; }

<a href="/sobre">Sobre</a>

Atributo: a[href]

Seleção por atributo segue a mesma regra do elemento, porém considerando que ele irá selecionar todos os elementos que possuam o atributo selecionado idependente do seu valor.

img[alt] {}

<img src="#" alt="Descrição">

Valor do atributo: a[href="#"]

Seleção por valor atributo segue a mesma regra da selação de atributo, porém considerando que ele irá selecionar todos os elementos que possuam o atributo selecionado e o valor selecionado.

a[href="http"]

Valor igual.

img[alt="#ParaCegoLer"]

<img src="" alt="#ParaCegoLer">

a[href^="http"]

Início do valor igual.

a[href^="http://"]

<a href="http://www.google.com">

a[href$=".br"]

Final do valor igual.

a[href$=".br"]

<a href="http://playground.com.br">

a[href*="http"]

Qualquer parte do valor igual.

a[href*="pt-br"]

<a href="http://pt-br.playground.com">

a[href~="http"]

Valores sepados por espaço.

.aplicativo[class~="tema-escuro"]

<div class"aplicativo tema-escuro"></div>

<div class"aplicativo-pwa tema-escuro"></div>

Concatenação: h1, p

Multiplas seleções de elementos.

h1, .titulo { font-family: sans-serif; }
<h1>...</h1>
<h2 class"titulo">...</h2>

Descendentes: li strong

Seleciona elementos descendente direto ou indiceto do elemento pai.

.nav li { background: red; }

<ul class="nav">
  <li>Filho</li>
  <ul>
      <li>Descendente</li>

Descendente: li > a

Seleciona elementos descendente direto do elemento pai.

.nav > li { background: red; }

<ul class="nav">
  <li>Filho</li>
  <ul>
      <li>Descendente</li>

Irmão: h1 ~ h2

Seleciona o irmão/paraleto do elemento.

h1 ~ h3 { background: red; }

<h1>...</h1>
<h3>...</h3>
<h2>...</h2>
<h3>...</h3>

Adjacente: h1 + p

Seleciona o irmão/parelelo direto do elemento.

h1 + h3 { background: red; }

<h1>...</h1>
<h3>...</h3>
<h2>...</h2>
<h3>...</h3>
